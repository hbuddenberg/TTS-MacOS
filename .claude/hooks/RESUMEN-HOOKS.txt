═══════════════════════════════════════════════════════════════════
🚀 INICIO RÁPIDO
═══════════════════════════════════════════════════════════════════

1. Habilitar TTS:
   
   export TTS_ENABLED=true

2. Iniciar Claude Code:
   
   claude-code

3. ¡Listo! Ahora todas las respuestas se leerán en voz alta.

═══════════════════════════════════════════════════════════════════
🎛️ CONFIGURACIÓN COMPLETA
═══════════════════════════════════════════════════════════════════

# Copiar y pegar en la terminal:

export TTS_ENABLED=true          # Habilitar TTS
export TTS_VOICE=monica          # Voz: monica, jorge, paulina, etc.
export TTS_RATE=175              # Velocidad: 100-300 WPM
export TTS_MAX_LENGTH=500        # Máximo de caracteres

# Opcional: Confirmación de prompts
export TTS_PROMPT_ENABLED=false  # true para habilitar
export TTS_PROMPT_VOICE=jorge
export TTS_PROMPT_RATE=200

═══════════════════════════════════════════════════════════════════
🔧 HACER PERMANENTE
═══════════════════════════════════════════════════════════════════

Para que la configuración se mantenga, agrega a ~/.zshrc:

echo 'export TTS_ENABLED=true' >> ~/.zshrc
echo 'export TTS_VOICE=monica' >> ~/.zshrc
echo 'export TTS_RATE=175' >> ~/.zshrc

Luego recarga:
source ~/.zshrc

═══════════════════════════════════════════════════════════════════
🎯 ALIAS ÚTILES
═══════════════════════════════════════════════════════════════════

Agrega a ~/.zshrc:

alias claude-tts='TTS_ENABLED=true claude-code'
alias claude-monica='TTS_ENABLED=true TTS_VOICE=monica claude-code'
alias claude-jorge='TTS_ENABLED=true TTS_VOICE=jorge claude-code'
alias claude-silent='TTS_ENABLED=false claude-code'

Uso:
claude-tts        # Con voz por defecto
claude-monica     # Con voz Monica
claude-jorge      # Con voz Jorge

═══════════════════════════════════════════════════════════════════
🐛 SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════════════

No se escucha nada:
  → Verifica: echo $TTS_ENABLED (debe ser "true")
  → Verifica volumen del sistema
  → Prueba: tts-macos "test" --voice monica

La voz no se encuentra:
  → Lista voces: tts-macos --list
  → Instala voces: System Preferences → Accessibility → Spoken Content

Hooks sin permisos:
  → chmod +x .claude/hooks/post-response
  → chmod +x .claude/hooks/user-prompt-submit
