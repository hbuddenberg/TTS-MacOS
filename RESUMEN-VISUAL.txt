┌──────────────────────────────────────────────────────────────────────┐
│  🚀 INSTALACIÓN EN 4 PASOS                                           │
└──────────────────────────────────────────────────────────────────────┘

    1️⃣  Descomprimir
         cd ~/Downloads
         tar -xzf mcp-tts-macos.tar.gz
         cd mcp-tts-macos

    2️⃣  Instalar
         ./install.sh

    3️⃣  Reiniciar Claude Desktop
         Cmd+Q para cerrar, luego abrir de nuevo

    4️⃣  ¡Probar!
         "Lee en voz alta: Hola mundo"

│  📊 ESPECIFICACIONES TÉCNICAS                                        │
└──────────────────────────────────────────────────────────────────────┘

    Lenguaje:          Python 3.10+
    Protocolo:         MCP (Model Context Protocol)
    Sistema operativo: macOS únicamente
    Dependencias:      mcp (Python package)
    Motor TTS:         macOS native 'say' command
    Formato audio:     AIFF
    Tamaño:            ~10 KB (comprimido)
    Licencia:          MIT (código abierto)
